# 📈 YouTube Stock Channel Sentiment Analysis for KOSPI Prediction

> 유튜브 주식 콘텐츠의 감성 분석을 통해 시장(KOSPI)을 선행하는 정보가 존재하는지를 검증하고 감성 지수 기반 투자 보조 서비스 가능성을 탐구한 프로젝트입니다.

팀원 : 우영서(3기), 김태연(4기), 김미정(5기), 강세진(5기), 이승화(5기), 장동혁(5기)

## 🧠 Project Motivation

최근 유튜브는 증권사 리포트, 경제 뉴스 못지않게
개인 투자자들의 핵심 정보 획득 채널로 자리 잡았습니다.

그러나 다음과 같은 문제가 존재합니다:
	•	주식 관련 유튜브 채널 과다 → 정보 선택의 어려움
	•	자극적인 콘텐츠로 인한 투자 판단 왜곡 가능성
	•	유튜브 정보가 시장을 선행하는지에 대한 검증 부재

👉 “유튜브 주식 콘텐츠의 감성이 실제 시장을 선행하는가?”
이 질문에서 프로젝트를 시작했습니다.

## 🎯 Objective
	•	주식 유튜브 채널의 텍스트(자막) 감성 분석
	•	감성 지수와 KOSPI 지수 간 시차(lead-lag) 관계 분석
	•	통계적으로 유의한 채널 식별
	•	감성 지수 기반 투자 보조 서비스 가능성 검증

## 🔍 Methodology
### 1. YouTube 채널 선정
	•	경제/주식 카테고리 상위 채널 탐색
	•	기준
	•	주식 관련 콘텐츠 중심
	•	업로드 주기 짧음
	•	조회수/구독자 수 상위

📌 주요 채널 예시
한국경제TV, 3PRO TV, 매일경제TV, MTN, 서울경제TV 등

### 2. 데이터 수집
	•	기간: 2024.06.01 ~ 2024.12.31
	•	국내 시장(KOSPI) 관련 영상만 선별
	•	YouTube API를 활용해 영상 자막(script) 수집

### 3. Text Preprocessing & Sentiment Analysis
	•	자막 → 문장 단위 분리
	•	512 토큰 이하로 chunking
	•	FinBERT 모델 활용
	•	금융 도메인 특화 감성 분석 모델
	•	출력: Positive / Neutral / Negative

➡️ 각 영상별 감성 점수를 평균 감성 지수로 변환

### 4. Sentiment ↔ Market Mapping
	•	KOSPI 전일 대비 상승 여부 이진화
	•	상승: 1 / 하락: 0
	•	감성 지수와 D-5 ~ D+5 구간의 주가 변동 매핑
	•	감성과 주가 간 선·후행 관계 분석

### 5. Statistical Analysis
	•	비율 검정 (H0 vs H1)
	•	H0: 감성으로 주가 방향을 맞출 확률 = 0.5
	•	H1: 감성으로 주가 방향을 맞출 확률 > 0.5
	•	채널별 · 시차별 정확도 및 p-value 분석

## 📊 Key Results & Insights

### ✅ 주요 발견
	•	일부 채널은 시장을 선행하는 감성 패턴을 보임
	•	특히 한국경제TV
	•	미래 방향성(D+1~) 예측에서 통계적으로 유의
	•	선행적 정보 전달 채널로 확인
	•	3PRO TV
	•	D-5 ~ D-3 구간에서 유의
	•	후행적 정보 전달 성향

### 📺 Traditional Media vs YouTube
	•	리포트/뉴스 대비
	•	유튜브 감성 지수의 선행성이 더 뚜렷
	•	단기(1개월)·중기(6개월) 모두 유사한 경향

### 💰 Backtesting (Investment Simulation)
	•	감성 지수 기반 전략:
	•	감성 +1 → 전액 매수
	•	감성 -1 → 전액 매도
	•	결과:
	•	전 구간에서 수익률 우세
	•	특히 하락장에서 방어 성능 우수

➡️ 감성 지수 기반 투자 전략의 실효성 확인
